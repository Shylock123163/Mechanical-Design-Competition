# 机械设计大赛团队资料库

团队协作项目，用于存储和展示机械设计大赛的所有资料。

## 在线访问

网站地址：`https://你的用户名.github.io/robot_web/`

## 项目结构

```
robot_web/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── main.js         # 交互脚本
├── files/              # 项目资料文件夹
│   ├── design/         # 设计图纸 (SolidWorks, CAD等)
│   ├── code/           # 程序代码
│   ├── docs/           # 技术文档
│   ├── media/          # 视频素材
│   ├── bom/            # 零件清单
│   ├── ppt/            # 演示PPT
│   └── meetings/       # 会议记录
└── images/             # 作品图片
```

## 如何贡献

1. Clone 仓库
```bash
git clone https://github.com/你的用户名/robot_web.git
```

2. 创建分支
```bash
git checkout -b feature/你的功能名
```

3. 提交更改
```bash
git add .
git commit -m "描述你的更改"
git push origin feature/你的功能名
```

4. 创建 Pull Request

## 文件上传说明

- 设计图纸放入 `files/design/`
- 程序代码放入 `files/code/`
- 技术文档放入 `files/docs/`
- 作品图片放入 `images/`

## 注意事项

- 大文件（>100MB）请使用 Git LFS
- 提交前确保文件命名规范
- 敏感信息不要上传
